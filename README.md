# Цифрова схемотехніка, розробка схем на Arduino у середовищі Proteus

У репозиторії містяться проекти які я писав під час вивчення дисціпліни "Цифрова схемотехніка та архітектура операційних систем"  
Проекти написані у середовищі розробки Arduino IDE на мові C++ та схемно реалізовані у Proteus

1. «7 сегментний індикатор»
2. «Чотиризначний 7-сегментний індикатор»
3. «LCD-дисплеї»
4. "Cвітлодіоди"
5. «Серводвигуни»
6. «Вимірювання температури»
7. «Світлофор»
8. «Робота з кнопками та асинхронним кодом»
9. «Командний замок»
